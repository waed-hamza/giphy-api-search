## About GIPHY app:
It's a fun project uses jQuery and AJAX! to make call to the GIPHY API and search for random GIFs.

<br/>


## How: 
 - Clone the repo 
 1. In treminal, type:

 ```
 - git clone https://github.com/waed-hamza/giphy-api-search.git
 - cd giphy-api-search
 - code .
 ```

 2. in VS code :
 **Go Live**
